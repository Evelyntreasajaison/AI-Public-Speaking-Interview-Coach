# Key Features
1.Real-time facial landmark detection using webcam input
2.Tracks eye contact, blink rate, head movement, and body posture
3.Computes an overall confidence score by combining multiple behavioral cues
4.Live feedback dashboard built with Streamlit
5.Designed to help users improve public speaking, interviews, and presentations

# How It Works
1.Captures live video through the webcam
2.Extracts facial landmarks and posture information
3.Analyzes confidence-related metrics such as gaze stability and posture alignment
4.Aggregates metrics into a confidence score
5.Displays real-time feedback and insights on a Streamlit dashboard
